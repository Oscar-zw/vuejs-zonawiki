# vuejs-zonawiki

## Project setup
```
$ git clone https://github.com/djdiego88/vuejs-zonawiki

$ cd vuejs-zonawiki

$ sudo docker-compose build

$ sudo docker-compose up -d
```

Visit:
```
http://localhost:8080
```
